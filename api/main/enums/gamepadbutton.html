Enumeration GamepadButton

The standard gamepad buttons. Only applies to controllers which are recognized as standard gamepad controllers. This enum contains generic enum names and some common aliases for Xbox and Playstation gamepads.

See https://w3c.github.io/gamepad/#dfn-standard-gamepad-layout
