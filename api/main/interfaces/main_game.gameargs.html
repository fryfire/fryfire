Optional backgroundColor

backgroundColor: undefined | string | Color

The background color. Defaults to black. Before a frame is rendered the canvas is filled with this color.

Optional pixelated

pixelated: undefined | boolean
  • Defined in src/main/Game.ts:32

Set to true for a pixelated game. image smoothing and image rendering is configured accordingly to this setting. Defaults is true when a fixed game size is specified, false if not.

Optional size

size: undefined | null | Size
  • Defined in src/main/Game.ts:26

Optional physical game size. When set then the canvas uses this exact size and aspect ratio but is still scaled up (by CSS) to fill as much of the screen as possible. When not specified then the physical size is dynamically calculated from the window size and device pixel ratio.
