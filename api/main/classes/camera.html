draw

  • draw(ctx: CanvasRenderingContext2D, width: number, height: number): boolean | void | (() => boolean | void)
    • Defined in scene/Camera.ts:593

    Draws this camera. The method can return a boolean which indicates if the scene is not finished yet and must be drawn continuously (for animations for example). The method can also return an optional function which is called after the child nodes are drawn so this can be used for post-drawing operations. This post-draw function then can again return an optional boolean which indicates that scene must be continuously draw itself.

    Parameters

    • ctx: CanvasRenderingContext2D

      The rendering context.

    • width: number

      The scene width.

    • height: number

      The scene height.

    Returns boolean | void | (() => boolean | void)

    Optional boolean to indicate if scene must be redrawn continuously (Defaults to false) or a post-draw function which is called after drawing the child nodes and which again can return a flag indicating continuos redraw.

focus

  • focus(target: CameraTarget, args?: FocusArgs): Promise<boolean>
    • Defined in scene/Camera.ts:427

    Stops following the currently followed target and focuses the given target. When no animation duration is given then this happens instantly. Otherwise the camera smoothly transitions over to the target. When the follow flag is set then the target is followed after focusing it.

    It is allowed to change the focus while the animation is still running. The previous animation is then canceled and the new animation transitions from the current position over to the target.

    Parameters

    • target: CameraTarget

      The target scene node to focus.

    • args: FocusArgs = ...

      Optional focus arguments.

    Returns Promise<boolean>

    True when focus was successfully set, false when the transition to the target was canceled by another focus call.
