Class Signal<T>

Light-weight and very fast signal/slot based event system. Just create a signal instance, connect slots (event listeners) to it and then let the signal emit values which are then send to all connected slots. Slots can be methods and it's easy to disconnect them again using the disconnect() method or calling the function returned by connect().
