formatNumber

  • formatNumber(value: number, options?: Intl.NumberFormatOptions & { locales?: string | string[] }): string
    • Defined in src/main/util/string.ts:110

    Formats a number to a string. The returned string is never in scientific notation so the string may get pretty long. NaN und infinite numbers are rejected because they can't be represented as a numerical string.

    Parameters

    • value: number

      The numeric value to convert.

    • Optional options: Intl.NumberFormatOptions & { locales?: string | string[] }

      Optional number format options. Defaults to english fullwide locale, not using number grouping and using 6 maximum fraction digits.

    The numerical string.
